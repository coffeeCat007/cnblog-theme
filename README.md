## 简单使用
1. 克隆到本地，运行`npm i`
2. 运行`npm run dev`
3. 复制`dist`目录的`css`和`js`内容到博客园相关设置里即可

## 自定义配色
在`themes/default`目录里，`base.less`可以设置4种配色和基本字体大小，默认配色是黑色主题。
